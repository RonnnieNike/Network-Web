Network


Network Layer: The internet or network or network layer is an abstraction layer supported by the IP (Internet Protocol) network protocol.
Ip is a network protocol that serves to connect the network. It is a packet protocol without a best-effort connection, therefore it does not guarantee reliability in terms of error control, flow control and congestion, made possible however by other protocols such as TCP and UDP or ICMP. ICMP communicates error or control messages. It is at the IP level and can also carry messages or requests and answers to requests, they are transmitted through IP datagrams
The PING command performs a reachability test by sending an ICMP echo request message to a host waiting for a reply, if the host does not respond, cannot be reached, then it is used to identify errors or network failures. It therefore serves as a glue for the various sub-networks at routing addressing level, allowing the exchange of information between them, therefore an interworking. Thus within an IP network, each interface is assigned a unique address, and can be calculated manually or by protocols such as DHCP or PPP.
This address is characterized by a subnetmask that allows to calculate if another IP address belongs to the same subnet or not.
